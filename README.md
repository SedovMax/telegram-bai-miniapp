# Telegram BAI Mini App

Мини-приложение Telegram для прохождения Шкалы тревоги Бека (BAI) и отслеживания динамики.

## Быстрый старт

1. Склонируй репозиторий в GitHub и подключи к Vercel (New Project).
2. Заполни переменные окружения из `.env.example`.
3. В Supabase выполни SQL из `supabase/schema.sql`.
4. Задеплой. В @BotFather добавь кнопку `web_app` на URL проекта.

## Телеграм
Бот уже у тебя есть. В меню бота добавь кнопку c типом Web App на URL деплоя Vercel.

## Dev
```bash
pnpm i
pnpm dev
```
